# ✨ Niccolò Miranda Clone  

A static recreation of Niccolò Miranda’s paper-inspired portfolio — exported from Webflow and rebuilt as a browser-ready project. This repo demonstrates how motion-driven design can be adapted into a fully offline static site.  

---

## 🚀 Overview  
This project is a **fully static portfolio landing page** inspired by Niccolò Miranda’s signature design style. It uses Webflow-exported HTML, CSS, images, and vendor libraries to deliver immersive scrolling and animated interactions — all without external dependencies .  

---

## 📖 About  
Why this project?  
- To explore how Webflow exports can be transformed into a developer-friendly static site.  
- To showcase **smooth animations, responsive layouts, and motion libraries** in action.  
- To provide a ready-to-use template for anyone experimenting with creative portfolio designs.  

---

## 📂 Project Structure  
- `index.html` — Main portfolio homepage  
- `test.html` — Verification page for CSS, image, and JS loading  
- `assets/` — Static assets and vendor libraries  
  - `css/` — Stylesheets  
  - `fonts/` — Custom fonts  
  - `images/` — Portfolio imagery & SVGs  
  - `vendors/` — Third-party scripts (GSAP, Locomotive Scroll, jQuery, etc.)  

---

## ✨ Key Features  
- Animated navigation & page transitions  
- Smooth scrolling powered by **Locomotive Scroll**  
- Motion library support with **GSAP**  
- Responsive, high-contrast layout  
- Offline-ready with local assets
